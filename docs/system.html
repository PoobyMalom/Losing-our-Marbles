About the System

This system combines software, firmware, electrical, and mechanical components to create a dynamic marble-based art machine. The user can draw or select an image, which is compressed and translated into a sorting matrix by our software. The firmware on an Arduino UNO controls servos, relays, and a DC motor to sort and arrange marbles by color into a physical frame, guided by a color sensor and solenoid mechanism. With an Archimedes screw for recycling marbles and a custom-built mechanical frame, this project seamlessly blends technology and creativity to turn digital designs into tangible art.
